# MEDRIN JOBS

## DESCRIPTION
Medrin Jobs is a modern and responsive web application that bridges the gap between job seekers and employers. It provides a user-friendly platform where : 
   - Job Seekers can search for jobs, apply for them and manage their professional profiles, including 
     uploading 
     resumes, cover letters and profile pictures.
   - Employers can post job vacancies, manage applications and shortlist or reject canditates.Employers can 
     post up two jobs for free after which posting additional jobs requires selecting a paid plan.


The platform is designed with ease of use in mind, featuring advanced search options with categories and 
location filters to streamline navigation for both job seekers and employers.

## TECHNOLOGIES USED
The project was built with the following technologies :
  1. Frontend Framework - Reacr Vite
  2. Styling - Tailwind CSS
  3. Routing - React Router Dom

## Features

### Job Seeker Features
- Job Search : Browse jobs by location, category, or keyword.
- Profile Management :  Create and manage a profile with personal details, resume, cover letter, and profile 
                        picture.
- Job Applications : Apply for jobs and track application statuses.

### Employer Features
- Profile Management : Create and manage a company/organization profile.
- Job Posting : Post job listings (first 2 free, subsequent posts require a paid plan).
- Application Management :  View and manage applications, shortlist candidates, and reject or approve 
                            applications.

### Global Features 
- Searches and Filters : Navigate seamlessly using filters for job categories, locations, and keywords.
- Responsive Design : Optimized for mobile and desktop devices.

## Installation

Follow these steps to set up the project locally:

1. Clone the repository :
    https://github.com/Joshndegwa13/medrin_jobs
2. Install dependencies :
    npm install
3. Start the development server
    npm run dev
4. Open your browser and navigate to :
    http://localhost:5173

## Usage

### For Job Seeker 
1. Sign up and create a profile with your resume, cover letter, and profile picture.
2. Use the search bar to find jobs by category, location, or keywords.
3. View job details and apply directly through the platform.

### For Employers/ Organisation
1. Sign up and set up your company profile.
2. Post job vacancies (first 2 free, subsequent posts require payment).
3. Manage applications by viewing applicant profiles, shortlisting, rejecting, or approving candidates.
